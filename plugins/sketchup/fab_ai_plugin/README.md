# Fab AI Plugin - 专业版 SketchUp 插件

## 🎯 插件简介

这是一个专为 **Fab（晶圆厂）设计院**开发的专业级 SketchUp 插件，集成AI技术，可大幅提升前期方案设计效率。

### ✨ 核心功能

#### 1. 🎨 草图转3D白模
- **输入**：手绘建筑透视图
- **AI识别**：自动提取建筑比例关系
- **输出**：正交多视图 + 3D 白模
- **特点**：30秒完成，保持比例精确

#### 2. 📐 红线参数化布局
- **输入**：建筑红线 + 项目参数
- **AI生成**：3个布局方案自动生成
- **输出**：完整的3D场景布局
- **特点**：智能优化，符合规范

## 🚀 快速开始

### 第一步：安装插件（2分钟）

**Windows 用户：**
```
1. 按 Win+R，输入：%AppData%
2. 进入：SketchUp\SketchUp 20XX\SketchUp\Plugins\
3. 把整个 fab_ai_plugin 文件夹拖进去
4. 重启 SketchUp
```

**Mac 用户：**
```
1. 按 Cmd+Shift+G，输入：
   ~/Library/Application Support
2. 进入：SketchUp 20XX/SketchUp/Plugins/
3. 把整个 fab_ai_plugin 文件夹拖进去
4. 重启 SketchUp
```

### 第二步：验证安装

打开 SketchUp，菜单栏应该出现：
```
插件 > Fab AI 设计助手
```

## 💻 功能演示

### 功能一：草图转3D白模

**操作步骤：**
1. 点击：`插件 > Fab AI 设计助手 > 草图转3D白模`
2. 点击上传区域（或点击"使用演示图片"）
3. 等待 AI 识别（2.5秒，显示进度）
4. 查看生成的四视图（正/侧/顶/轴测）
5. 点击"生成 SketchUp 3D 白模"
6. ✅ 查看场景中的 3D 模型

**生成内容：**
- 🏢 主厂房（洁净室）80m×60m×15m + 技术夹层3m
- ⚡ 动力站 30m×25m×12m + 设备层2m
- 🧪 化学品库 20m×15m×8m
- 💨 气体站 12m×10m×6m
- 🏢 综合办公楼 35m×18m×18m（5层）
- 🌉 连廊 4m×25m×4m

**关键特性：**
- ✓ 真实的进度条显示
- ✓ SVG 绘制的多视图预览
- ✓ 详细的比例分析数据
- ✓ 自动生成复杂建筑组合

---

### 功能二：红线参数化布局

**操作步骤：**
1. 点击：`插件 > Fab AI 设计助手 > 红线参数化布局`
2. 点击"导入红线"（会显示红线信息）
3. 设置项目参数：
   - Fab类型：12寸晶圆产线
   - 洁净室面积：8000㎡
   - 月产能：60,000片
   - 扩展预留：30%
   - 勾选配套设施
4. 点击"智能生成布局方案"
5. 等待生成（3秒，显示进度）
6. 对比三个方案，选择一个（推荐方案B）
7. 点击"生成选中方案的3D模型"
8. ✅ 查看场景中的完整布局

**生成内容：**
- 📐 用地红线边界（150m×100m）
- 🛣️  内部道路系统（主干道12m + 次干道8m）
- 🌳 绿化区域（前侧8m + 后侧10m）
- 🏢 主厂房 60m×50m×15m
- 🏢 办公楼 30m×15m×15m
- ⚡ 动力站 25m×20m×12m
- 📦 仓储区 30m×20m×10m

**三个方案对比：**
| 方案 | 面积利用率 | 容积率 | 特点 |
|------|-----------|--------|------|
| 方案A | 78% | 1.2 | 紧凑型，节约用地 |
| **方案B** | **82%** | **1.45** | **平衡型，流线优秀** ⭐ |
| 方案C | 75% | 1.3 | 扩展型，预留50% |
